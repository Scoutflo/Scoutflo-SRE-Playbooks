## All Contributors

We recognize all contributions, including:
- Playbook authors
- Documentation improvements
- Bug reports and fixes
- Feature suggestions
- Code reviews

### How to Get Listed

1. Make a contribution (any size counts!)
2. Your GitHub profile will automatically appear in the [Contributors graph](https://github.com/Scoutflo/scoutflo-SRE-Playbooks/graphs/contributors)
3. Significant contributors will be added to the Core Contributors section

## Recognition

We believe in recognizing all forms of contribution:

- **Code/Content**: Playbooks, documentation, workflows
- **Design**: Improving structure and organization
- **Documentation**: README, guides, examples
- **Community**: Answering questions, reviewing PRs
- **Ideas**: Feature requests, improvement suggestions
