# Roadmap

> Our vision for SRE Playbooks in 2026 and beyond.

## Q1 2026 (Completed)

- [x] Initial release with 163 playbooks
- [x] AWS restructuring to service-based folders
- [x] Sentry playbooks (25)
- [x] Proactive monitoring playbooks (121)
- [x] GitHub Pages documentation site
- [x] Open source community setup

## Q2 2026 (In Progress)

- [ ] **Dual-format playbooks**: Add CLI equivalents to all playbooks
- [ ] **GitHub Pages enhancements**: Search, navigation, dark mode
- [ ] **50 more K8s playbooks**: Expand coverage
- [ ] **GCP playbooks**: Google Cloud Platform support
- [ ] **Azure playbooks**: Microsoft Azure support

## Q3 2026 (Planned)

- [ ] **Interactive troubleshooting**: Decision tree wizard
- [ ] **Playbook versioning**: Track changes per playbook
- [ ] **Community contributions**: Streamlined contribution workflow
- [ ] **Localization**: Support for multiple languages
- [ ] **API access**: Programmatic access to playbooks

## Q4 2026 (Planned)

- [ ] **AI agent integrations**: Pre-built connectors
- [ ] **Metrics correlation**: Link playbooks to monitoring systems
- [ ] **Custom playbook builder**: UI for creating playbooks
- [ ] **Enterprise features**: Private playbooks, SSO

## Future Vision

### 1000+ Playbooks
Cover every major incident scenario across all cloud providers.

### Universal SRE Standard
Establish a community-driven standard for SRE playbooks.

### AI-Native
Full integration with AI agents for automated incident response.
