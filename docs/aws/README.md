**Critical (P0):**
- EC2 instance down
- RDS database unavailable
- Production Lambda errors

**High (P1):**
- High latency issues
- Partial service degradation
- Security alerts

**Medium (P2):**
- Performance warnings
- Cost anomalies
- Configuration drift
